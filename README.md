# showUrl: an unbrowser that shows the URL that an application asked to open

Various applications contain buttons that open URLs, often without a provision to copy the URL for separate use.

Sometimes I click the button by accident.  
Sometimes I want to go to the contents of the link.

Often, I want a choice of which browser the link opens in.

Here is my bodge to respond to these preferences.  
It is recognisable as a browser app in (at least) MATE.
It displays the link in a copyable form in a terminal window.
Then I can copy the link and paste it into the browser window of my chosing.
